Concepto de Git

Git es un sistema de control de versiones distribuido ampliamente utilizado en el desarrollo de software. Permite a los desarrolladores realizar un seguimiento de los cambios en sus archivos y colaborar eficientemente en proyectos en equipo.

Principales comandos de git

  • GIT INIT: Este comando funciona para iniciar un proyecto con GIT
  • GIT ADD: Este comando funciona para todos los documentos en el area de reparacion
  • GIT CONFIG: El comando "git config" en Git se utiliza para configurar y obtener información sobre la configuración de Git en tu sistema. Puedes usarlo para establecer opciones globales, específicas de un repositorio o locales a un proyecto en particular.
  • GIT COMMIT: Permite dejar un comentario dentro de Git
  • GIT STATUS:Permite visualizar el estado actual del archivo
  • GIT LOG:Permite visualizar el historial de lo realizado
  • GIT BRANCH:Permite visualizar en que rama nos encontramos
  • GIT -b BRANCH "NAME":Permite cambiar el nombre de la rama en el que nos encontramos
  • GIT REMOTE ADD ORIGIN "URL":Permite entrelazar el GIT con el GIThub con el link que nos otorga
  • GIT CONFIG --GLOBAL USER.NAME: Nos permite ingresar el nombre con el que nos identifica
  • GIT CONFIG --GLOBAL USER.EMAIL:Nos permite ingresar el correo con el que vamos a estar identificados
  • GIT PUSH -U ORIGIN MAIN:Este comando sirve para activar el repositorio o el branch
  • GIT CHECKAUT:Este comando funciona para desplazarse entre las ramas creadas por git branch
